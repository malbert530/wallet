# Финансовое приложение: управление кошельками

## Описание

Приложение предоставляет REST‑API для управления балансом кошельков:
- выполнение операций (пополнение/снятие);
- получение текущего баланса.

### Для запуска проекта использовать команду
```bash
   docker compose up -d
```

## Доступные эндпоинты

### Выполнение операций с кошельком
**Метод:** `POST`  
**URL:** `/api/v1/wallets`  
**Тело запроса:**
```json
{
  "walletId": "UUID",
  "operationType": "DEPOSIT" | "WITHDRAW",
  "amount": 1000
}
```
**Ответ:**
```json
{
  "id": "UUID",
  "balance": 1000
}
```

**Метод:** `GET`  
**URL:** `/api/v1/wallets/{walletUUID}`

**Ответ:**
```json
{
  "id": "UUID",
  "balance": 1000
}
```
